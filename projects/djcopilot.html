DJ CoPilot

Description

IN PROGRESS!!!

In collaboration with my friend Nina Boord (who's a DJ herself), we're developing a web application that assesses song compatibility in order to help DJs brainstorm when deciding what to mix and in what order. The backend, given a list of songs, currently returns for each song all other compatible songs in order of strength of compatibility based on key features such as BPM, melodic structure, and percussive profile. We hope to add more compatibility criteria in the near future such as song key, informed by user interviews with our friends who are experienced DJs.

When complete, this application will have user profile management, allowing users to log in and leverage songs from previous searches in future results. The UI will be developed using React.js, and the Python backend will use Django to communicate with the frontend. In order to house the data, we'll be using a PostgreSQL database, and as for hosting the site itself, we'll leverage the AWS platform along with Docker to gain experience with containerization and cloud hosting/management.
